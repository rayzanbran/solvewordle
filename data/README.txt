DATA folder:
This folder holds data used by the program. The location of each of these files is stored in consts.py in the main
program folder. 

IMPORTANT: The program is counting on this data to be in an accessible place. If any of these files is moved, the
corresponding path must be changed in consts.py or the program will not function!

counts.csv - Count of each letter in the wordle dictionary.
letterscores.csv - Calculated score assigned to each letter.
valid-wordle-words.txt - The entire Wordle dictionary as of 2023 (thus, it may be inaccurate)
